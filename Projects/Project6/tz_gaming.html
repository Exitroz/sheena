# Check multicollinearity using VIF (Variance Inflation Factor)
from statsmodels.stats.outliers_influence import variance_inflation_factor

# Define a function to calculate VIF for each feature
def calculate_vif(df, features):
    vif_data = pd.DataFrame()
    vif_data["feature"] = features
    vif_data["VIF"] = [variance_inflation_factor(df[features].values, i)
                        for i in range(len(features))]
    return vif_data

# Features to check for multicollinearity
features = ['imppat', 'clpat', 'ctrpat']
print(calculate_vif(train_data, features))

# Now, creating a model without 'clpat'
lr_mc2 = LogisticRegression()
lr_mc2.fit(train_data[['imppat', 'ctrpat']], train_data['click'])

# Recalculate VIF without 'clpat'
features.remove('clpat')
print(calculate_vif(train_data, features))

In the initial model, both imppat and clpat have very high VIF values, far exceeding the common threshold of 10, which indicates a significant level of multicollinearity. This high level of multicollinearity can lead to several issues:

Unreliable Coefficient Estimates: The regression coefficients of highly collinear variables can become very sensitive to minor changes in the model. This can make the model unstable and the interpretation of the coefficients questionable. Inflated Standard Errors: Multicollinearity can inflate the standard errors of the coefficients, leading to wider confidence intervals and making it harder to determine if a variable is statistically significant. Compromised Predictive Performance: While multicollinearity doesn't affect the model's ability to predict or fit the data, it can make the model less reliable for understanding the impact of individual features.

In the modified model, after removing clpat, the VIF values for both remaining explanatory variables (imppat and ctrpat) drop well below the threshold of 5, indicating that the remaining variables have little to no multicollinearity. This reduction in multicollinearity has several positive impacts:

More Stable Coefficients: The regression coefficients are likely to be more stable and reliable for interpretation, as they are less influenced by changes in the model or data. Improved Significance Testing: Lower VIF values mean that the standard errors of the coefficients are also likely to be lower, making it easier to determine the statistical significance of the variables. Clearer Interpretation: With reduced multicollinearity, the model's coefficients can be more confidently interpreted in terms of the independent effect of each explanatory variable on the response variable.

By identifying and removing highly collinear variables, you can improve the interpretability and reliability of your model, making it a more useful tool for understanding the relationships between variables.

def calculate_performance(data, prediction_column, actual_column, total_impressions, conversion_rate, clv, cpm_cost):
    # Calculate the number of targeted impressions based on the prediction column
    targeted_impressions = data[data[prediction_column] == True].shape[0]
    
    # Calculate expected conversions
    expected_conversions = targeted_impressions * conversion_rate
    
    # Calculate expected revenue
    expected_revenue = expected_conversions * clv
    
    # Calculate total cost (including the cost for 20M impressions)
    total_cost = (total_impressions / 1000) * cpm_cost
    
    # Calculate profit
    profit = expected_revenue - total_cost
    
    # Calculate ROME
    rome = (profit / total_cost) * 100  # ROME as a percentage
    
    return profit, rome
